The Hospital Management System is a Java-based, menu-driven console application designed to simulate real-world hospital administration workflows. It manages key hospital operations such as patient records, doctor details, appointment scheduling, pharmacy management, laboratory services, staff management, and billing using Object-Oriented Programming principles.

This project demonstrates the practical application of abstraction, inheritance, collections, and structured control flow in Java.

Patient Management

Doctor Management

Appointment Scheduling

Book appointments between patients and doctors

Generate appointment details with fixed consultation charges

Pharmacy Module

Maintain medicine inventory

Purchase medicines with quantity selection

Automatically calculate medicine costs

Laboratory Services

View available lab tests

Book multiple lab tests per patient

Add lab service costs to billing

Staff Management

Manage nurses, workers, and security staff

Filter staff records based on designation

Billing System

Generate consolidated bills for patients

Include medicines, lab tests, and appointment charges

Apply 18% tax (GST) automatically

Display detailed itemized bill summary

Hospital Facilities Overview

Add and display available hospital facilities
